Thank you for downloading Penguin.

This is a semi realistic 3d model and animations of an Penguin. 
-Models folder contains AdeliePenguinModelLowPoly.FBX, AdeliePenguinModelMiddlePoly.FBX, AdeliePenguinModelTwoLODs.FBX, AdeliePenguinHumanoidModel.FBX, AdeliePenguinHumanoidModelTwoLODs.FBX,
AfricanPenguinModelLowPoly.FBX, AfricanPenguinModelMiddlePoly.FBX, AfricanPenguinModelTwoLODs.FBX.
-Animations folder contains FBX files of animations.
Walk animations are compatible with root motion.
-Textures folder contains textures.
These are main contents of this package.

Demo folder contains an example use of the Penguin model. 
-Demo/Scenes folder contains demo scene.
-Demo/Prefabs folder contains prefabs that are used in the demo scene.
-Demo/Scripts folder contains scripts that are used in the demo scene.
-Demo/Animator folder contains animator controller that is used the prefab.
